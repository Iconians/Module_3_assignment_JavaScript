#Build these projects out and submit them

**Utility Library (Node.js)**
Build a utility library with 5 useful functions in Node.js:
- Create `utils.js` with functions like: capitalize, reverse string, random number generator, etc.
- Create `test.js` that uses and tests all functions
- Export functions using `module.exports`
- Run with `node test.js`

**Rock Paper Scissors (Terminal)**

- Use readline for input
- Run with `node rps.js`
